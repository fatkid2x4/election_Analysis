# election_Analysis

## Project Overview

A Colorado Board of Elections employee, named Tom, had given me the following task.

1.Calculate the Total number of votes cast
2.Get a  complete list of candidates who received votes
3.Need Total number of votes each candidate received
4.Get Percentage of votes each candidate won
5.Determine the winner of the election based on popular vote

## Resources
-Data Source: election_results.csv
-Software: Python 3.9.13 and Visual Studio Code 1.74.1

## Summary
The analysis of the election show that:
There were 369,711 votes cast.
There were 3 candidates
  a. Charles Casper Stockholm received 23.0% of the votes for 85,213 total votes
  b. Diana DeGette received 73.8% of the votes for 272,892 total votes
  c. Taymon Anthony Doane received 3.1% of the votes for 11,606 total votes
Diana DeGette was declared the winner.
